Git 别名是用于简化 Git 命令的快捷方式，只有以下 1 个，所有 Git 仓库都能使用：
| 别名名称 | 完整原始命令 | 核心功能（对你的作用） |
|-------|--------|------------|
| **git acp** | !f() { git add . && git commit -m "$1" && git push --force-with-lease origin $(git symbolic-ref --short HEAD); }; f | 一键完成「添加所有文件 → 提交（带备注） → 安全强制推送到当前分支」 <br>对你的 cs-study 仓库：推送到 main 分支，实现快速备份 |

# 二、Bash 别名（共 11 个，永久生效）
Bash 别名是用于简化终端命令的快捷方式，所有终端会话都能使用（因为配置在 ~/.bashrc 中，~/.bash_aliases 为空）。
你的「临时别名」其实是当前会话加载的永久别名，不是手动临时配置的，关闭终端后重新打开，这些别名依然存在。
按功能分类如下，方便你记忆：
## 1. ls 系列（文件查看，最常用）
| 别名名称 | 完整原始命令 | 核心功能（对你的作用） |
|-------|--------|------------|
| **ls(`自带`)** | ls --color=auto | 查看文件，带颜色区分（目录蓝色、可执行文件绿色等） |
| **ll(`自带`)** | ls -alF | 查看所有文件（含隐藏文件）+ 详细权限 / 大小 + 类型标记（目录结尾加 /） |
| **la(`自带`)** | ls -A | 查看所有文件（含隐藏文件，排除 . 和 ..） |
| **l(`自带`)** | ls -CF | 按列查看文件（不含隐藏文件） + 类型标记（简洁版） |
## 2. 保护类（防止误操作，核心安全）
| 别名名称 | 完整原始命令 | 核心功能（对你的作用） |
|-------|--------|------------|
| **cp** | cp -i | 复制文件时，若目标文件已存在，提示确认覆盖 |
| **mv** | mv -i | 移动/重命名文件时，若目标文件已存在，提示确认覆盖 |
| **rm** | rm -i | 删除文件时，提示确认（关键：防止手滑删光系统文件）（重要：防止误删 rm -rf /*） |
## 3. 搜索类（带颜色，方便查找）
| 别名名称 | 完整原始命令 | 核心功能（对你的作用） |
|-------|--------|------------|
| **grep(`自带`)** | grep --color=auto | 搜索文件内容，匹配的关键词高亮显示 |
| **fgrep(`自带`)** | fgrep --color=auto | 快速搜索固定字符串（不解析正则表达式） |
| **egrep(`自带`)** | egrep --color=auto | 支持扩展正则表达式的搜索 |
## 4. 工具类（提升效率）

# 三、SSH Agent 自动启动配置
```bashrc
# 检查 SSH 代理是否已运行（通过环境变量 SSH_AUTH_SOCK 判断）
if [ -z "$SSH_AUTH_SOCK" ]; then
    # 启动 SSH 代理
    eval "$(ssh-agent -s)"
    # 加载私钥（路径替换为你的实际私钥路径，默认是 ~/.ssh/id_ed25519）
    ssh-add ~/.ssh/id_ed25519 > /dev/null 2>&1
fi
# === 配置结束 ===
```

# 四、Bash 历史记录优化（保留更多记录，带时间戳）
```bashrc
HISTSIZE=100000                   # 内存中保留的历史命令数（默认 1000）
HISTFILESIZE=200000               # 历史文件中保留的命令数（默认 2000）
HISTTIMEFORMAT="%F %T "           # 历史记录带时间戳（格式：年-月-日 时:分:秒）
HISTCONTROL=ignoredups:ignorespace # 忽略重复命令和以空格开头的命令（避免敏感命令记录）
shopt -s histappend               # 新命令追加到历史文件，而非覆盖
PROMPT_COMMAND="history -a; $PROMPT_COMMAND"  # 每次命令执行后，立即写入历史文件
```
